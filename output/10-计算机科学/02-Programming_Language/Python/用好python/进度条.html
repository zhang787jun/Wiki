1. Tqdm

Tqdm 是一个快速，可扩展的Python进度条，可以在 Python 长循环中添加一个进度提示信息，用户只需要封装任意的迭代器 tqdm(iterator)。

1.1. 安装

pip安装：

pip install tqdm

1.2. 使用

from tqdm import tqdm,trange

# 1. 
for i in tqdm(range(10000)):
     sleep(0.01)

# 2. 
for i in trange(100):
        sleep(0.1)

# 3.
pbar = tqdm(["a", "b", "c", "d"])
for char in pbar:
    pbar.set_description("Processing %s"% char)


# 4. 
with tqdm(total=100) as pbar:
    for i in range(10):
        pbar.update(10)

# 5. 
pbar = tqdm(total=100)
for i in range(10):
    pbar.update(10)
pbar.close()
